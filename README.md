# Executive E-Commerce Performance Dashboard & Predictive Analytics

An interactive web application designed for e-commerce store managers and executives to monitor financial performance, track sales metrics, and leverage machine learning for future revenue forecasting.

## 🚀 Live Demo
[View Live App](https://your-username.streamlit.app) <!-- Replace with your actual Streamlit URL -->

## ✨ Features
* **Financial KPIs**: Real-time tracking of Net Revenue, Gross Profit, and Return Rate.
* **Sales Analytics**: Dynamic visualization of top-selling products by volume.
* **Profitability Insights**: Interactive line charts tracking monthly profit margins.
* **Predictive Forecasting**: Integrated Machine Learning model that forecasts next month's revenue based on historical data.

## 🛠️ Tech Stack
* **Frontend/Framework**: Streamlit
* **Data Manipulation**: Pandas, NumPy
* **Data Visualization**: Plotly Express
* **Machine Learning**: Scikit-Learn
